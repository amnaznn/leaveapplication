# Leave Application System

This is a comprehensive Employee Leave Application System built with **Laravel**. 

## 🚀 Features
* **Employee Dashboard:** View personal leave history and submit new leave requests.
* **Admin Portal:** Manage all employee leave applications.
* **Role-Based Access:** Secure authentication for employees and admins.

## 🛠️ Prerequisites
* **PHP** (8.2 or higher)
* **Composer**
* **Node.js & NPM**
* **Database** (SQLite)
