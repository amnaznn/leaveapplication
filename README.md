# Leave Application System

This is a comprehensive Employee Leave Application System built with **Laravel**. It allows employees to submit leave requests and enables administrators to manage those requests efficiently.

## 🚀 Features
* **Employee Dashboard:** View personal leave history and submit new leave requests.
* **Admin Portal:** Manage all employee leave applications (approve/reject/delete).
* **Role-Based Access:** Secure authentication for both employees and administrators.

## 🛠️ Prerequisites
Before running this application, ensure you have the following installed:
* **PHP** (8.2 or higher)
* **Composer**
* **Node.js & NPM**
* **Database** (MySQL or MariaDB)
